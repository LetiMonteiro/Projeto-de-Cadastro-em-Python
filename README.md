## Sobre o Projeto

Esse projeto é muito especial para mim, porque foi o primeiro que desenvolvi 100% sozinha 💙

Foi nele que coloquei em prática tudo o que venho aprendendo, enfrentando desafios, errando, aprendendo e melhorando a cada etapa e cada aula. Cada parte do código carrega um pouco da minha evolução como desenvolvedora e se Deus quiser saberei fazer projetos muito maiores.

O sistema é um cadastro de usuários via terminal, um proejto simples, com validação de dados como email e senha, além do armazenamento das informações em arquivo JSON, o que tornou o projeto ainda mais completo e funcional.

Mais do que um projeto técnico, ele representa um passo importante na minha jornada na programação — mostrando que, com dedicação e curiosidade, eu consigo construir algo do zero ✨
